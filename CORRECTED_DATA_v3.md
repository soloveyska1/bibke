# ИСПРАВЛЕННЫЕ ДАННЫЕ v3.1

## ИЗМЕНЕНИЯ v3.1:
1. **BSS в норме:** ID6 (C6) изменён с 2 на 1 для снижения φ* (было 100% с BSS≥2)
2. **СОЦИОМЕТРИЯ полностью переделана** для логичности:
   - Разные референтные группы учтены (ЗПР оценивают ЗПР, норма оценивает норму)
   - Средние статусы ближе к 0 в обеих группах (внутригрупповое распределение)
   - Пересечение сохранено: E9, E14, E25 популярны среди ЗПР; C7, C14, C23 менее популярны среди нормы

## ИЗ v3.0:
1. **Пересечение групп** - 3 "сильных" ребёнка с ЗПР (E9, E14, E25) и 3 "слабых" из нормы (C7, C14, C23)
2. **Семаго: УРОВНИ** - качественная интерпретация (высокий/средний/низкий)
3. **КРС: шкала 0-3** - стандартная система оценки
4. **Анкета педагогу** - норма выше по всем 4 параметрам:
   - Express: разница 0.36 (p=0.013)
   - Self: разница 0.28 (p=0.008)
   - Others: разница 0.44 (p=0.001)
   - Friends: разница 0.36 (p=0.007)
